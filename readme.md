This Project analyzes air pollution datasets ("pm10", "pm25", "no", "no2", "nox", "co", "o3", and "so2") for multiple global cities.
It filters data by pollutant and date, and generates four plot per pollutant:

1. Time Series (concentration vs hour)

2. Bar plots (mean, median, mode, max, min)

3. Histogram (distribution)

4. Box plot (quartiles)

Data Sources: OpenAQ platform (CSV files)

Tools: Python, Pandas, Matplotlib
